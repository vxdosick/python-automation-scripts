# Web Parser

A simple Python script that parses a static web page and extracts:

- Main title (h1)
- Secondary titles (h2)
- Images (src and alt)

## Usage

1. Activate virtual environment:
```bash
source venv/bin/activate
```

2. Run the parser:
```bash
python parser.py
```

3. Check the output in parsed_data.txt

## Dependencies

Python 3

requests

beautifulsoup4
