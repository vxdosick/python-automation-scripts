# File Organizer (Python Automation)

A simple Python script that organizes files in a folder by their extensions.

## Features
- Sorts files into categories:
  - Images
  - Documents
  - Videos
  - Audio
  - Archives
  - Other

## Usage
```bash
python file_organizer.py
```

Enter the path to the folder you want to organize.

## Tech
Python 3
os, pathlib, shutil
